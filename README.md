# timinh/frankenphp
A Frankenphp dockerfile ready for symfony (or other php) projects.

### Contains the following extensions : 
 - pdo
 - intl
 - mysqli
 - gd
 - ldap
 - gettext
 - calendar
 - ctype
 - session
 - dom
 - pdo
 - pdo_mysql
 - curl
 - zip

### Contains the following utility programs : 
 - composer
 - git
 - nano
 - vi
